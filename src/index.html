National Contiguity with a Force Directed Graph

A freeCodeCamp Project

This project was created for the freeCodeCamp curriculum. It makes an ajax request using the axios library and bind the data to a generated svg graph utilizing The D3.js data visualization library.

The json data contains nodes (countries), and links (connections). A force-graph was created with each link displaying geographic adjacency. Moving the cursor over each flag will display the name of the country. Dragging the flag will pull that node's connections along with it.

User Story Requirements:

  1. I can see a Force-directed Graph that shows which countries share borders
  2. I can see each country's flag on its node
